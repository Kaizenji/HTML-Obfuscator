HTML Obfuscator API

This API obfuscates your HTML code by removing scripts, comments, and applying obfuscation techniques to make the code less readable.

Endpoint

GET /html-obfuscator?code=<your-html-code>

How to Use

  1. URL-encode your HTML code to be obfuscated.
  2. Send a GET request to /html-obfuscator with the code query parameter set to your encoded HTML.
  3. Receive the obfuscated HTML as plain text in the response.

Example Request

GET /html-obfuscator?code=%3C!DOCTYPE%20html%3E%3Chtml%3E%3Chead%3E%3Ctitle%3EMy%20Page%3C%2Ftitle%3E%3C%2Fhead%3E%3Cbody%3E%3Ch1%3EHello%2C%20world!%3C%2Fh1%3E%3Cp%3EThis%20is%20a%20short%20HTML%20example.%3C%2Fp%3E%3C%2Fbody%3E%3C%2Fhtml%3E

Example Response

<!-- obfuscated html here -->

Features

  • Removes <script> tags from the HTML.
  • Removes HTML comments.
  • Obfuscates the HTML code to make it less readable.
  • Easy integration via simple GET requests.

Notes

  • Make sure to URL-encode the HTML when sending it as a query parameter.
  • Obfuscation is done automatically on the server side.
  • Large HTML input may take longer to process due to network latency.
